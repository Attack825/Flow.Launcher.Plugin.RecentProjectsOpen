# Flow.Launcher.Plugin.VscodeOpen

This plugin is designed to enhance your workflow by providing quick access to your previously opened projects in Visual Studio Code (VSCode). If you find yourself frequently switching between different projects or need to revisit recently accessed codebases, this plugin is here to simplify that process.

**How to Use:**

1. Install the plugin from the plugin marketplace or manually.

2. Once installed, simply invoke your launcher and type the designated keyword to activate the VSCode History Projects Launcher.

3. A list of your recent projects will be displayed, allowing you to select the one you want to open. Press enter, and voila! You're back to coding in no time.

**Compatibility:**

This plugin is compatible with the latest version of Visual Studio Code and is regularly updated to ensure optimal performance and compatibility.
